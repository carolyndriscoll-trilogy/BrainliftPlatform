# DOK1 Grading App

Reusable grading tool for brainlifts. Each brainlift gets its own URL.

## Deployment to Replit

1. Create new Replit → Choose "Vite + React" template
2. Delete the default files
3. Upload and unzip `dok1-grading-app.zip` (or drag all files)
4. Click "Run"

## URLs

- Home: `your-replit-url.repl.co/`
- Alpha Schools: `your-replit-url.repl.co/grading/alpha-schools`
- Knowledge Rich: `your-replit-url.repl.co/grading/knowledge-rich-curriculum`

## Adding a New Brainlift

1. Create a new JSON file in `src/data/` (e.g., `my-new-brainlift.json`)
2. Follow this structure:

```json
{
  "title": "My Brainlift Name",
  "description": "Short description",
  "summary": {
    "totalFacts": 15,
    "meanScore": "4.5",
    "score5Count": 10,
    "contradictionCount": 2
  },
  "facts": [
    { "id": "1", "category": "Research", "fact": "Fact text here", "score": 5, "contradicts": null },
    { "id": "2", "category": "Policy", "fact": "Another fact", "score": 4, "contradicts": "Cluster Name" }
  ],
  "contradictionClusters": [
    {
      "name": "Cluster Name",
      "factIds": ["2", "3"],
      "claims": ["Claim 1", "Claim 2"],
      "tension": "Description of the tension",
      "status": "Flagged"
    }
  ],
  "readingList": [
    { "id": 1, "type": "Twitter", "author": "@handle", "topic": "Topic", "time": "3 min", "facts": "What it covers", "url": "https://..." }
  ]
}
```

3. Import it in `src/App.jsx`:

```javascript
import myNewBrainlift from './data/my-new-brainlift.json'

const brainlifts = {
  'alpha-schools': alphaSchools,
  'knowledge-rich-curriculum': knowledgeRich,
  'my-new-brainlift': myNewBrainlift,  // Add this line
}
```

4. Save and the new URL is live: `/grading/my-new-brainlift`

## Categories

Available categories (color-coded):
- `Regulatory` (purple)
- `External Benchmarks` (blue)
- `Research` (teal)
- `Louisiana` (green)
- `Alpha Internal` or any other (orange)

## Reading List Types

- `Twitter` (blue)
- `Substack` (coral)
- `Blog` (purple)
- `Research` (teal)
