The Core UX Problem

Right now the top section is trying to do four jobs at once:

Identify the artifact

Communicate its status

Provide navigation

Offer actions

All four are visually co-equal, which creates cognitive noise.

The fix is hierarchy, not removal.

Target Mental Model

The user should read this top section in this order, without effort:

What is this artifact?

What state is it in?

What mode am I currently in?

What can I do next?

Everything below enforces that sequence.
